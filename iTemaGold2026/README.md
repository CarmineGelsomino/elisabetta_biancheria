# iTemaGold2026

Tema WooCommerce sviluppato da **OC Corporation**.

## Obiettivi implementati
- AccessibilitĂ : skip-link, focus visibile, semantica HTML5, supporto reduced motion.
- SEO: title-tag nativo, markup semantico, JSON-LD Organization.
- GEO: meta geografici standard (`geo.region`, `geo.position`, `ICBM`).
- Sicurezza: hardening header HTTP (`nosniff`, `SAMEORIGIN`, `Referrer-Policy`) e rimozione `the_generator`.
- CompatibilitĂ  WordPress 2026: supporto editor moderno, HTML5, responsive embeds, WooCommerce support completo.
- Personalizzazione: pannello **Aspetto > Personalizza** con opzioni per colori globali, font principale, larghezze layout e raggi pulsanti.

## Installazione
1. Copia la cartella `iTemaGold2026` in `wp-content/themes/`.
2. Attiva il tema da **Aspetto > Temi**.
3. Installa/attiva WooCommerce.
4. Imposta i menu in **Aspetto > Menu** (`primary`, `footer`).
5. Configura stile e branding da **Aspetto > Personalizza > iTemaGold2026: Stile globale**.
